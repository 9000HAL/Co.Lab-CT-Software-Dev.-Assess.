Why JSONPlaceholder?

JSONPlaceholder was chosen for this project as it is a free online REST API that allows for quick & simple fetching of data for testing/prototyping. It provides a wide range of data types including users, todos, posts, and comments, so it is versatile to simulate API calls for various frontend tasks. Its comprehensive documentation & community support further make it an ideal choice for those gaining experience with API integrations -- like me!
